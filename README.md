# 说明
manosaba

魔法少女的魔女审判

中日双语补丁，同时显示中文与日文

# 适用版本

1.0.6

# 使用方法
选项1：使用任意x64DLL注入工具 将manosaba.dll注入到manosaba.exe中

选项2：将version.dll和manosaba.dll放入manosaba.exe同目录下，进入游戏将自动注入
